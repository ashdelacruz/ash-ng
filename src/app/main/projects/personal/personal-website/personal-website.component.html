Presentation

I built the frontend originally using Angular 16. Notable features include session management with JWTs, robust form handling, and dynamic themeing.
I also used Capacitor to create iOS and Android runtimes for the project, but they are not available on the App or Play store because that costs money and I ain't tryna do all that.

Persistence

Backend uses SpringBoot 3. Notable features include JWT and token url validation, SpringSecurity authentication framework, and custom UserDetails implementation.
Initially I set up the Email Service using AWS SES to allow for a custom ashdelacruz.com from email. I set up SQS queues and SNS topics to handle bounces and complaints, but AWS denied my request for production access. So I moved to Gmail SMTP, without a custom domain.

Database

I'm using MongoDB as my database. Coming from a SQL-heavy IBM stack, I was hesitant to make the switch to NoSQL. But the setup is quick, intuitive, and most importantly, free.
The project provides a UI for User Moderation, accesible only to Moderator or Admin level users, althought there is a demo version available. The project also contains a User Settings section to allow users to modify their own info.

Host

The whole application is hosted on a RaspberryPi 4 Model B in my living room. Both frontend and backend are running as Docker containers, and are launched via Docker Compose.
The frontend container uses an nginx image to run the Angular project, with custom config to enable SSL.
