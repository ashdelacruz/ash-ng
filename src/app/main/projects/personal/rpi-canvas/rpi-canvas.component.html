Raspberry Pi Canvas

This project is an RGB LED display controleld by a Raspberry Pi.
Here is the original project from Reddit user u/gonnabuysomewindows

My initial goal was to create an album cover display. It works via Airplay 2 so you can connect to it from any iOS device. You can Airplay to the Pi Canvas alongside any other Airplay 2 enabled speaker or headphones, and it will display the album cover of whatever song is playing in Apple Music.

I added a secondary "Art Display" mode which just displays an image or series of images, stored locally on the Pi. You can control which images will be shown, the speed at which they're shown, and upload new images directly from your iPhone.

I created a suite of iOS shortcuts to control the Pi Canvas over wifi. It uses the "Run script over SSH" shortcut to do most of the work, so it requires you to set up an SSH key from the Pi.

Overall, this project exposed me to a lot of different technologies. All of the coding was done on the command line and I had previous experience working on Linux systems, both of which helped me understand the Raspberry Pi and how to run servers on it. I learned how to create bluetooth and Airplay connections using MQTT and shairport-sync. I'd never worked on hardware before though so I had to learn about eletronics and soldering for this project.
